# InstaCV
# Ruana Soares

Aplicativo com OpenCV para criar efeitos em imagens e video.

- Escolha uma imagem para começar aplicando os filtros conforme o menu.

- Input via teclado (letras maiúsculas) e números de 0 - 9.


            *****MENU***** 
    
            ESC - fechar janela 
    
            B - Blur (GaussianBlur) 
    
            C - detector de bordas Canny 
    
            D - detector de bordas Sobel 
    
            E - escrever na imagem 
    
            F - Dilatacao 
    
            G - Grayscale 
    
            H - erosao 
    
            I - Mouse CallBack - Sticker - charuto 
    
            J - Mouse CallBack - Sticker - oculos 
    
            M - Mouse CallBack - Sticker Nariz 
    
            O - Sticker com OverlayImage e coordenadas via teclado 
    
            R - Sticker com Coordenada via input Nariz 
    
            S - Salvar imagem 
    
            V - Execucao de video e manipulacao do trackbar 
    
            X - Salvando um frame 
    
            0 - Retira o ruido da imagem 
    
            1 - Cartoon 
    
            2 - Pencil 
    
            3 - Paraiso artificial - interpolacao 
    
            4 - sun 
    
            5 - Fire 
    
            6 - Purple 
    
            7 - Blue 
    
            8 - Green day 
    
            9 - Ajusta a gramatura da imagem
