# InstaCV
# Ruana Soares

Aplicativo com OpenCV para criar efeitos em imagens e video.


# Input:
1. Permitir ao usuário escolher a foto para carregar 

2. Permitir fazer a captura de vídeo pela webcam

# Execução: 
Dois tipos de operações serão possíveis:

1. Colar stickers – a partir de uma base de no mínimo 5 stickers diferentes (sprites
com transparência), selecionar pela interface (clique do mouse) e posicioná-los
acima da foto (o posicionamento pode ser feito via mouse ou teclado)

2. Aplicar filtro – a partir de uma base de no mínimo 10 efeitos diferentes,
selecionar pela interface (clique do mouse) e com isso a foto será alterada de
acordo com o filtro.

# Output:
salvar a foto alterada. No caso do vídeo, salvar um frame.
